# Fonts Directory

Place your Telugu font files here:

## Required Fonts:

1. **NotoSansTelugu-Regular.ttf** - Noto Sans Telugu (Regular)
2. **NotoSerifTelugu.ttf** - Noto Serif Telugu
3. **Panchganga.ttf** - Panchganga font
4. **Peddana-Regular.ttf** - Peddana (Regular)

## Where to Download:

1. **Noto Fonts** (Google Fonts):
   - Visit: https://fonts.google.com/noto/specimen/Noto+Sans+Telugu
   - Visit: https://fonts.google.com/noto/specimen/Noto+Serif+Telugu
   - Download and extract TTF files

2. **Panchganga Font**:
   - Visit: https://fonts.google.com/specimen/Panchganga
   - Download and extract

3. **Peddana Font**:
   - Visit: https://fonts.google.com/specimen/Peddana
   - Download and extract

## Installation:

1. Download the font files
2. Place them in this `/public/fonts/` directory
3. Make sure the file names match exactly:
   - `NotoSansTelugu-Regular.ttf`
   - `NotoSerifTelugu.ttf`
   - `Panchganga.ttf`
   - `Peddana-Regular.ttf`

The application will automatically load these fonts when generating cards.
